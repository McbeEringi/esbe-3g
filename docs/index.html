ESBE 3G v2.0

Features

  • 軽量かつ確実な動作
  • 滑らかで自然な影
  • 輝度ベースかつ彩度の変化するトーンマップ
  • 揺れる草花と水面
  • 新しい水面反射
  • and some more…

Comparations

乱数使用回数(水面)9回14回
トーンマップガンマ補正+トーンカーブガンマ補正+彩度補正+トーンカーブ
揺れるブロック草 葉 サトウキビ 作物2G+花 キノコ ネザー系草 ベリー系　etc…
